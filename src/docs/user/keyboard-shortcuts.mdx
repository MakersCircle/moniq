# Keyboard Shortcuts

Moniq is designed for high-efficiency data entry. Use these shortcuts to speed up your workflow and manage your finances without ever touching your mouse.

## Global Shortcuts

- **`Alt + N`**: Open the **New Transaction** modal (defaulting to Expense).
- **`Shift + E`**: Open modal or switch to **Expense**.
- **`Shift + I`**: Open modal or switch to **Income**.
- **`Shift + T`**: Open modal or switch to **Transfer**.

> [!NOTE]
> Global `Shift` shortcuts are only active when you are not currently typing in a text field. They will also switch the active tab if the modal is already open.

## Transaction Entry Modal

The entry modal is optimized for rapid, sequential data logging.

### Navigation & Tabs
- **`Alt + 1`** (or `Cmd/Ctrl + 1`): Switch to **Expense** tab.
- **`Alt + 2`** (or `Cmd/Ctrl + 2`): Switch to **Income** tab.
- **`Alt + 3`** (or `Cmd/Ctrl + 3`): Switch to **Transfer** tab.
- **`Tab`**: Move focus to the next input field.
- **`Shift + Tab`**: Move focus to the previous input field.

### Data Selection
- **Dropdowns**: When a dropdown (Method or Category) is focused, simply start typing the name to jump to it. For example, type "ca" to select "Cash".
- **Smart Date Entry**: The date field is highly flexible. You can type:
  - Full dates: `ddmmyyyy`, `dd/mm/yyyy`, or `dd-mm-yyyy`.
  - Shorthand: Type `ddmm` (like `0512` for Dec 5th) and it will assume the current year.
  - Ultra-shorthand: Type just the day (`d`) to pick that day in the current month.
  - The field automatically cleans up symbols and formats everything to `DD/MM/YYYY` when you move to the next field.
- **Date Picker**: When the date field is focused, use the **Arrow Down** key to open the calendar, then use arrow keys to navigate.

### Saving Transactions
- **`Enter`**: Save the transaction and close the modal. This works in all fields, including **Notes** (use `Shift + Enter` for new lines in notes).
- **`Cmd + Enter`** (or `Ctrl + Enter`): **Save & Stay**. This saves the current transaction and shows a "Saved" indicator, but keeps the modal open for your next entry. It clears the amount, note, and category selection but preserves the **Date**, **Payment Method**, and **Tab** to make bulk logging effortless. Focus is automatically returned to the **Amount** field.

## Navigation & UI

- **`Esc`**: Close any open modal or dropdown.
- **`Backspace`** (on selected item): Delete or clear selection in compatible lists.
